## pre-requisites

- node
- npm
- react-native

## how to run

1. clone the repo

```bash
 git clone https://github.com/one04-eg/transactions.git
```

2. navigate to the project directory

```bash
 cd transactions
```

3.  install dependencies

```bash
 npm install
```

4. run the app, if you have android device connected ( emulator or real device)

```bash
 npm run android
```
